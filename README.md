# marikaimu-ardona.github.io

Personal portfolio of **Marikaimu Chris Ardoña**, Senior Front End Developer.

Live site: https://marikaimu-ardona.github.io

## About

A hand-coded, dependency-free static site showcasing selected work, technical skills, and experience.

- Semantic HTML5
- A small custom CSS design system (no frameworks)
- Vanilla JavaScript for navigation and scroll reveals
- Responsive and accessible (reduced-motion aware)

## Structure

```
.
├── index.html        # Page content
├── css/styles.css    # Design system and layout
└── js/main.js         # Mobile nav, scroll reveal, footer year
```

## Development

No build step. Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deployment

Served via GitHub Pages from the `main` branch.
